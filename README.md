# sales-opportunity-dashboard
Power BI dashboard analyzing sales opportunities, channels, salespersons, and win rates using DAX and interactive visuals.
1. Overview

This project provides a comprehensive analysis of sales opportunities across different channels and sales teams. It highlights performance patterns, win/loss trends, and revenue drivers to support better sales decisions.

2. Dataset

Source: Excel/CSV mock dataset

Key Fields: Opportunity ID, Stage, Channel, Salesperson, Expected Revenue, Status, Close Date, Region.

3. Tools Used

Power BI

DAX (Data Analysis Expressions)

Excel/CSV for initial data preparation

Power Query for cleaning and transformations

4. Data Cleaning & Modeling

Removed duplicates and handled missing values

Standardized date and category fields

Created relationships between fact and dimension tables

Implemented a star-schema model for optimal performance

Built calculated columns and measures to enhance analytics

5. DAX Measures

Key DAX measures used in the dashboard include:

Win Rate %

Total Opportunities

Total Expected Revenue

Conversion Rate

Average Deal Size

Opportunities by Stage

Revenue by Channel

6. Key Insights

Identified top-performing sales channels driving the highest revenue

Highlighted salespersons with the strongest conversion rates

Exposed drop-off points in the opportunity pipeline

Showed comparison of won vs lost deals across time

Revealed revenue concentration across regions and channels

7. Dashboard Features

Interactive slicers for dynamic analysis

KPI cards for quick summary metrics

Funnel chart showing opportunity progression

Bar and line charts for performance tracking

Channel and salesperson-level breakdowns

Clean, user-friendly layout for easy interpretation

8. How to Use the Dashboard

Download the .pbix file from this repository.

Open it using Power BI Desktop.

If necessary, reconnect to your local dataset.

Use slicers and filters to explore insights interactively.
